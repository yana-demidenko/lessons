Основные теги html
1) Структура Web-страницы
<html></html> Начало и конец страницы
<head></head> Описание страницы,содержащее различную информацию не отображаемую в теле документа.Здесь распологается тег имени страницы и теги для поисковых машин.
<title></title> Имя страницы
<body></body> Определяет видимую часть страницы
2) Aтрибуты тела документа Значение атрибутов
<body background="URL"> Фоновое изображение
<body bgcolor="#RRGGBB"> Цвет фона документа, используя значения цвета в виде RRGGBB red,blue,#FF0000 и др.
<body text="#RRGGBB"> Цвет текста документа, используя значения цвета в виде RRGGBB
<body link="#RRGGBB"> Устанавливает цвет гиперсылок, используя значения цвета в виде RRGGBB
<body vlink="#RRGGBB"> Устанавливает цвет гиперсылок на которых вы уже побывали, используя значения цвета в виде RRGGBB
<body alink="#RRGGBB"> Устанавливает цвет гиперсылок при нажатии
3) Теги для форматирования текста Значение атрибутов
<h?></h?> Создает заголовок (уровни от 1 до 6)
<h? align="*"></h?> Загаловок с выравниванием left, right, center
<p></p> Создает новый параграф
<p align="*"></p> Выравнивает параграф относительно одной из сторон документа left, right, center, justify
<br> Вставляет перевод строки
<center></center> Выравнивание по центру
<div align="*"> Форматирование больших блоков текста страницы, также используется в таблицах стилей left, right, center, justify
4) Теги для форматирофания шрифта Значение атрибутов
<b></b> Жирный
<i></i> Курсив
<u></u> Подчеркнутый
<sup></sup> Верхний индекс - пример: a2
<sub></sub> Нижний индекс - пример: a2
<font size=?></font> Размер шрифта в пределах от 1 до 7
